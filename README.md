## 🚀 Getting Started

### 🔧 Installation

1. Clone the portfolio_v1 repository:

```sh
git clone https://github.com/jasper-th-wang/portfolio_v1
```

2. Change to the project directory:

```sh
cd portfolio_v1
```

3. Install the dependencies:

```sh
► npm install
```

### 🤖 Running portfolio_v1

```sh
► npm run start
```
